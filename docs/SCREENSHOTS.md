# Screenshot Gallery / Galeria zrzutów ekranu

EN: All screenshots were regenerated from the running Docker environment and current application state. Most application screens are captured with the `ADMIN` account so that the full navigation bar is visible. The notifications screen is captured from the author perspective because workflow notifications are addressed to the user affected by the event.

PL: Wszystkie zrzuty ekranu zostały wygenerowane ponownie z działającego środowiska Docker i aktualnego stanu aplikacji. Większość ekranów aplikacji wykonano z konta `ADMIN`, aby widoczny był pełny pasek nawigacji. Ekran powiadomień wykonano z perspektywy autora, ponieważ powiadomienia workflow trafiają do użytkownika, którego dotyczy zdarzenie.
